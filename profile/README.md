# ITU BIAI Lab

**Biomedical Imaging & AI Laboratory** · Istanbul Technical University
*Director: Assoc. Prof. Isa Yildirim*

We develop deep learning methods for medical imaging—focusing on radiation dose reduction, self-supervised learning, and open-source tools.

---

## Research

| Area | Focus |
|------|-------|
| **Low-Dose CT** | Diagnostic-quality imaging at reduced radiation |
| **Self-Supervised Learning** | Leveraging unlabeled medical data |
| **Digital Breast Tomosynthesis** | Open-source simulation & reconstruction |
